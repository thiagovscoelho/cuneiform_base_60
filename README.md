# cuneiform_base_60
Converter between decimal and cuneiform base 60, using Unicode characters, with 𒑊 as a zero. See it [here](https://thiagovscoelho.github.io/cuneiform_base_60/).

After publishing this, I realized that a webpage doing the exact same thing (in nearly just the same way I wanted it, with the Unicode and the zero and the spaces) was actually available [here](https://www.leskoff.com/s01764-0), but only in one direction, whereas this version can convert back to decimal as well. Also, I used different Unicode characters for the numerals 4, 5, 7, 8, and 9 (and those containing them) because I wanted it to look closer to the illustration on [Wikipedia](https://en.wikipedia.org/wiki/Babylonian_cuneiform_numerals).
