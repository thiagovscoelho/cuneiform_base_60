# cuneiform_base_60
Converter between decimal and cuneiform base 60, using Unicode characters, with 𒑊 as a zero. See it [here](https://thiagovscoelho.github.io/cuneiform_base_60/).

After publishing this, I realized that a webpage doing the exact same thing (in the exact same way I wanted it, with Unicode and the zero and the spaces) was actually available [here](https://www.leskoff.com/s01764-0), but only in one direction, whereas this version can convert back to decimal as well.
