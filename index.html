How it reads unspaced text

The parser tries the longest valid numeral first. That lets a two-sign digit such as 11 be recognized as one base-60 place rather than as separate digits 10 and 1.

Place-value rule

  1. Each cuneiform numeral represents one digit from 0 through 59.
  2. Moving one place left multiplies the value by 60.
  3. For example, 1 · 60² + 0 · 60 + 0 = 3600.
